# Redmi 9T で Picture Uploader を USB からインストールする手順

Redmi 9T（MIUI）は「端末情報」「ビルド番号」ではなく、**「デバイス情報」→「MIUIバージョン」** で開発者オプションを出します。

---

## ステップ1: 開発者向けオプションを表示させる

| 順番 | 操作 |
|------|------|
| 1 | **設定** アプリを開く |
| 2 | **「デバイス情報」** をタップ（※「端末情報」ではない。一覧の上の方にあることが多い） |
| 3 | **「MIUIバージョン」** の行を **7回連続でタップ** |
| 4 | 「デベロッパーになりました」や「開発者になりました」と表示されたらOK |

**見つからない場合:** 設定のトップで **検索** を使い、「デバイス情報」または「MIUI」で検索してみてください。機種・MIUIバージョンによっては **「マイデバイス」** → **「すべてのスペック」** の画面で **「MIUIバージョン」** を7回タップする場合もあります。

---

## ステップ2: 開発者向けオプションを開く

| 順番 | 操作 |
|------|------|
| 1 | **設定** に戻る |
| 2 | **「追加設定」** をタップ |
| 3 | 一覧を **一番下までスクロール** |
| 4 | **「開発者向けオプション」** をタップ |

**「追加設定」が見つからない場合:** 設定トップの検索で「追加設定」や「開発者」で検索。

---

## ステップ3: USB 関連をすべて ON にする

開発者向けオプションの画面で、次の3つを **ON** にします。

| 項目 | 説明 |
|------|------|
| **USBデバッグ** | ON → 確認ダイアログが出たら「OK」 |
| **USB経由でインストール** | ON（名称は「Install via USB」のことも） |
| **USBデバッグ（セキュリティ設定）** | ON（機種によってはここでSIMやMiアカウントを求められる場合あり） |

※「USB経由でインストール」をONにすると、**SIMを挿入していないとオフに戻る機種**があります。その場合は下記「方法B: APKをコピーしてインストール」を使ってください。

---

## ステップ4: PC と接続してインストール

1. Redmi 9T を **USBケーブル** で PC に接続する（**データ転送できるケーブル**であること）
2. スマホに **「USBデバッグを許可しますか？」** が出たら **「許可」** をタップ（「このコンピュータを常に許可」にチェック推奨）
3. PC で PowerShell を開き、次を実行:

```powershell
cd "c:\Users\yoich\docker-projects\Picture_Uploader"
.\adb-devices.ps1
```

4. 一覧に **デバイスが1行表示されていたら**、続けて:

```powershell
.\install-release.ps1
```

---

## うまくいかないとき

### 「USB経由でインストール」がすぐオフに戻る（SIM未挿入など）

→ **adb を使わずにインストール**します。

1. PC の `app\build\outputs\apk\release\app-release.apk` を、USBの「ファイル転送」でスマホの **Download** フォルダなどにコピー
2. スマホで **ファイル** アプリ → **Download** → **app-release.apk** をタップ
3. 「不明なアプリのインストール」を許可（求められたら、使っているアプリを選択して許可）→ インストール

### 「追加設定」や「デバイス情報」の場所が違う

→ 設定アプリの **検索**（画面右上の虫眼鏡など）で「開発者」「USBデバッグ」「デバイス情報」「MIUI」などで検索すると、該当メニューにたどりつけます。

---

## まとめ（Redmi 9T だけ）

1. **設定** → **デバイス情報** → **MIUIバージョン** を **7回タップ**
2. **設定** → **追加設定** → **開発者向けオプション**（一覧のいちばん下）
3. **USBデバッグ** / **USB経由でインストール** / **USBデバッグ（セキュリティ設定）** を ON
4. USB接続 → 許可 → `.\adb-devices.ps1` → `.\install-release.ps1`

SIMやセキュリティで「USB経由でインストール」が使えない場合は、APKをコピーして端末からインストール。
